## 1. frontend
- Vue.js
- Code dependencies
    - vue, vue-router,
    - vuex, vuex-router-sync,
    - axios, vue-axios,

Install dependencies:
```bash
$ npm install
```
Run web application:
```bash
$ npm run dev
```
Then, open http://localhost:8080 on browser

## 2. backend
- Node.JS, mongoDB, socket.io
- Code dependencies: express, mongoose
- Server host: http://localhost:3000 default

Install dependencies:
```bash
$ npm install
```
Run server:
```bash
$ npm start
```
